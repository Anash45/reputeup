# ReputeUp - Figma to HTML Project

This project converts Figma designs to HTML using Tailwind CSS.

## Project Structure

```
reputeup/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Custom CSS styles
├── js/
│   └── main.js         # Custom JavaScript
├── assets/
│   ├── images/         # Image files
│   ├── fonts/          # Custom fonts
│   └── icons/          # Icon files
└── README.md           # Project documentation
```

## Technologies Used

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript

## Getting Started

1. Open `index.html` in your browser
2. Or use a local server (e.g., XAMPP, Live Server)

## Development

- Tailwind CSS is loaded via CDN for quick prototyping
- Add custom styles in `css/style.css`
- Add custom scripts in `js/main.js`
- Place images, fonts, and icons in their respective `assets/` folders
